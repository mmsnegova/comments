# Comments

---

**Описание проекта**

Наш проект "Comments" представляетс собой блок с комментариями.
Блок содержит

- форму добавления комменатрия с валидацией
- список комментариев

**Технологии**

- HTML
- SASS
- SCC
- JavaScript
- Webpack
